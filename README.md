# vue-music-test

基于Vue2+axios制作的网易云播放器（非常简陋）

css自己写的，所以不是很好看。

歌曲封面需要获取自己的token，其他的音乐播放和列表都可以正常使用。

api获取地址：[ALAPI](https://www.alapi.cn/)

歌曲api：[网易云音乐 NodeJS 版 API (neteasecloudmusicapi.vercel.app)](https://neteasecloudmusicapi.vercel.app/#/)
